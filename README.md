# langl

langl: arc grains looper
by @burgess
based on angl by @tehn
engine: glut @artfwo

load files via param menu

E2 to switch modes
K2 to modifies

SPEED
- K2 then touch arc to
set speed to zero

LOOP
- K2 move loop position

PITCH
- K2 coarse control

VOLUME, DENSITY, SIZE,
JITTER, SPREAD
- K2 fine control
